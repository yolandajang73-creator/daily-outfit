# 点淘 AI 每日穿搭 Demo

这是一个用于讨论“点淘 AI 内容导购 / 每日穿搭决策台”的静态 HTML 原型。

## Demo 入口

打开 `index.html` 即可预览。

## 核心体验

- iOS App 风格的每日穿搭首屏
- 根据用户画像、今日场景、衣橱已有比例生成穿搭判断
- 推荐理由分为：
  - 结合你这个人
  - 搭配方法依据
  - 需要避开的坑
- 支持场景切换、穿搭卡切换、喜欢反馈、完整理由抽屉

## 当前边界

- 图片使用本地样例素材
- 用户画像与推荐理由为原型数据
- 未接入真实商品、库存、价格、交易和退货数据

## 建议讨论问题

1. “每日穿搭”是否能成为点淘新的日常打开理由？
2. 推荐理由里，“用户理解 + 搭配方法论 + 衣橱证据”的表达是否足够可信？
3. 下一步应优先接入衣橱购买记录、搭配方法卡，还是商品缺口推荐？
